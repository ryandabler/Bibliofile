How Does This Work?

Creators and Works Lists

The Creators and Works sections list all the entries in our database. Users can scroll down the list to find an entry they are interested in knowing more about or contributing data to. If the entry does not exist, clicking the "New" button in the toolbox on the left side will display a form to enter in the creator's or work's name. From there the user will see the details page for the newly created entry.

Creator and Work Details

The details page lists all information currently collected about the item in question. To add, modify, or delete information, click the "Edit" button in the left toolbox. This will display a large number of interactive elements that allow all info pieces to be changed.

Make sure before navigating away to click "Save" in the toolbox or all changes will be reverted. Clicking "Cancel" will revert all changes without leaving that page, and "Delete" will delete the entry and navigate back to the main listing.

Works

On the works details page, you have the option of adding a wide variety of information. Each section has the general properties:

Titles
Enter in the title and its language
Conbtributors
Anyone who has contributed to a work (author, translator, editor, etc.). Specify what role the person took in the creation of the work. Adding the contributor will display a dropdown, where you can click an existing author or manually enter a new one.
Contents
List the table of contents, including its kind (e.g., essay, chapter), number, and name
References
If the work references in some way another work, list the kind of reference (e.g., main in-text reference, footnote, artistic adaptation) and the title. If you choose the title from a drop down, a link will be created to that work; if you don't choose from the dropdown, you will create the work you listed behind the scenes.
Identifiers
Add any kind of number that is used to catalog a work (ISBNs, DOIs, Opus number, etc.)
Links
Any website that has information or a way to view the work should be added

Conversation

Generate a reading list by choosing a work from the dropdown that appears as you begin typing. Because of how the data is looked up on the server, it is important to select an item from the dropdown window.

A list of works that are tied together through their references field will be displayed. If the list is shorter than expected, that is because not enough works have been linked together. You can help expand the list by adding new works or fleshing out the references sections for works that already exist.
